# NotesManager

В проекте используется:
    Angular v17.0.7.
    RxJs v7.8.0

## Запуск приложения

1. В терминале в папке приложения последовательно ввести команды  
  1.1. npm i  
  1.2. ng s
2. В браузере открыть `http://localhost:4200/`

## Деплой приложения

Доступен по ссылке: https://notesmanages.onrender.com
